Ubuntu20.04开启与关闭图形界面

2022-03-25
Linux-LinuxUbuntu

关闭图形界面

sudo systemctl set-default multi-user.target

开启图形界面

sudo systemctl set-default graphical.target

输入完以上指令后，进行重启

sudo reboot

或

sudo shutdown -r now

shutdown命令简介

shutdown [选项] [时间] [警告信息]

选项

-r 关闭系统并立即重新启动。

-h 关闭系统后不重新启动。

-k 并不真正关机，只是送警告信号给每位登录者〔login〕。

-n 快速关闭系统，不经过init程序关机。

-f 快速关闭系统，重新启动时不进行磁盘检测。

时间

now 立刻

hh : mm 在hh时mm分

+m 在m分钟后

警告信息

警告信息会发送给所有用登录用户。

hello
